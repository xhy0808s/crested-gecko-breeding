🦎 시작하기

크레스티드 게코 관리 시스템을 효과적으로 사용하는 방법을 알아보세요

1단계: 계정 생성

먼저 계정을 만들고 로그인하세요. 무료로 시작할 수 있으며, 필요에 따라 Premium으로 업그레이드할 수 있습니다.

  • 이메일과 비밀번호로 회원가입
  • 게스트로 시작하여 기능 체험
  • Premium 플랜으로 고급 기능 이용

2단계: 개체 등록

크레스티드 게코 개체를 등록하고 기본 정보를 입력하세요.

  • 개체 이름, 성별, 세대 정보 입력
  • 모프 정보 선택 (복수 선택 가능)
  • 개체 이미지 업로드
  • 부모 개체 정보 입력 (F2 이상)

3단계: 혈통 관리

등록한 개체들의 혈통 관계를 확인하고 관리하세요.

  • 혈통 트리로 가족 관계 시각화
  • 조상/후손 트리 확인
  • 혈통 이미지 저장 및 공유

4단계: 모프 계산

부모 개체의 모프를 기반으로 자손의 모프 확률을 계산하세요.

  • 부모 모프 선택
  • 자손 모프 확률 계산
  • 유전 법칙 적용
  • 위험한 조합 경고

검색 및 필터

이름, 세대, 성별, 카테고리로 개체 검색

통계 분석

세대별 개체 수, 상태별 분포 확인

이미지 관리

개체 이미지 저장 및 재등록 기능

프리미엄 기능

AI 예측, 고급 분석, 번식 계획

사용 팁

  • 정기적으로 데이터를 백업하세요
  • 개체 등록 시 상세한 정보를 입력하세요
  • 혈통 트리를 활용하여 번식 계획을 세우세요
  • Premium 플랜으로 더 많은 기능을 이용하세요

지금 시작하세요!
