📱 아이폰에서 사용하기

아이폰에서 앱처럼 사용하는 방법을 안내합니다

1 웹 서버 시작하기

Python 사용 (가장 간단)

터미널/명령 프롬프트에서 다음 명령어를 실행하세요:

# 현재 폴더에서 웹 서버 시작 python -m http.server 8000

또는 Python 3가 설치되어 있다면:

python3 -m http.server 8000

Node.js 사용

Node.js가 설치되어 있다면:

# npx로 간단한 서버 실행 npx http-server -p 8000

Live Server (VS Code)

VS Code를 사용한다면:

1. Live Server 확장 프로그램 설치 2. index.html 파일 우클릭 3. "Open with Live Server" 선택
💡 팁: 서버가 시작되면 터미널에 "Serving HTTP on 0.0.0.0 port 8000" 같은 메시지가 표시됩니다.

2 아이폰에서 접속하기

IP 주소 확인

컴퓨터의 IP 주소를 확인하세요:

# Windows ipconfig # Mac/Linux ifconfig

보통 192.168.x.x 형태의 주소입니다.

Safari에서 접속

아이폰 Safari에서 다음 주소로 접속:

http://[컴퓨터IP]:8000 예시: http://192.168.1.100:8000
⚠️ 주의: 컴퓨터와 아이폰이 같은 Wi-Fi 네트워크에 연결되어 있어야 합니다.

3 앱 설치하기

Safari에서 홈 화면 추가

1. Safari 하단의 공유 버튼 탭

2. "홈 화면에 추가" 선택

3. 앱 이름 확인 후 "추가" 탭

✅ 완료! 이제 홈 화면에서 앱처럼 사용할 수 있습니다.
