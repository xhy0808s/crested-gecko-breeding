## 🦎 **주요 프로젝트**

### 🚀 **크레스티드 게코 브리딩 관리 시스템**
> **파충류 번식을 위한 완벽한 디지털 솔루션**

**주요 기능:**
- ✅ **개체 등록 및 관리** (F1~F5 세대)
- ✅ **모프 계산기** (유전 확률 계산)
- ✅ **가족 관계도** (4세대 시각화)
- ✅ **해칭 관리** (온도 기반 예측)
- ✅ **실시간 동기화** (PC ↔ 모바일)
- ✅ **Firebase 클라우드 백업**

**기술적 특징:**
- 🔄 **실시간 동기화**: Firebase Firestore 기반
- 📱 **모바일 최적화**: PWA 지원
- 🎨 **현대적 UI/UX**: 네이버 스타일 디자인
- 🔒 **데이터 안전성**: 클라우드 백업
- ⚡ **오프라인 지원**: 로컬 저장소

**배포 링크:**
- 🌐 **Netlify**: https://crested-gecko-breeding.netlify.app
- 📱 **GitHub Pages**: https://xhy0808s.github.io/crested-gecko-breeding/
